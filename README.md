# Personal Portfolio

This is a **personal portfolio website** made with **HTML, CSS, and JavaScript**.  
It showcases projects, skills, and contact information.

---

## Features

- Responsive **portfolio website** for desktop and mobile.  
- Sections include **Home, About, Projects, and Contact**.  
- Interactive navigation and smooth scrolling.  
- Showcases your **projects and skills** in an organized way.

---

## Technologies Used

- HTML  
- CSS  
- JavaScript
